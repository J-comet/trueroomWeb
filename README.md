### 주요기능

1. 회원가입 [ 재직자 / 구직자 구별 ] 
2. 로그인 [ 이메일저장 ] 
3. 비밀번호 변경, 내정보 수정 [ 재직자 / 구직자 정보 차별화 ]
4. 기업리뷰 게시판 
  - 검색
  - 신고
  - 페이징
  
5. 일반 게시판 
  - 다중 사진 업로드
  - 페이징
  - 댓글
  - 댓글 페이징
  
6. 문의하기 [ 이메일 - 네이버SMTP ]
7. 관리자페이지 [ 회원관리(계정 정지/해지), 게시글, 문의글 관리 ]
